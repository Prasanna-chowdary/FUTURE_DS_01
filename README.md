# 📊 Business Sales Performance Analytics

## 📌 Project Overview
This project analyzes business sales data to identify key insights such as revenue trends, top-selling products, category performance, and regional sales distribution.

## 🛠 Tools Used
- Power BI / Excel
- Data Analysis Techniques
- Data Visualization

## 📂 Dataset
The dataset contains sales transactions with the following fields:
- Order ID
- Order Date
- Region
- Category
- Product
- Sales
- Profit

## 📊 Dashboard Features
- Total Revenue, Orders, and Profit KPIs
- Monthly Revenue Trend Analysis
- Top-Selling Products
- Category-wise Sales Distribution
- Region-wise Performance Analysis

## 🔍 Key Insights
- Technology category generates the highest revenue
- West region performs better compared to others
- A few products contribute significantly to total sales
- Sales show steady growth over time

## 💡 Recommendations
- Focus on low-performing regions like South and Central
- Promote top-selling products to increase revenue
- Improve marketing strategies for Furniture and Office Supplies
- Maintain inventory for high-demand products
